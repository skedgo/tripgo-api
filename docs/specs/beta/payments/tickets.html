Complete Booking Flow

Expected flow with tickets and external payment

  • Compute trip to get a booking option (as usual)
  • Press book and retrieve tickets
  • Attempt to book sending selected tickets
  • The response includes payment options (only one at this stage), including the selected tickets and the corresponding price
  • Use the corresponding url to retrieve the ephemeral key and stripe publishable api key (see docs)
  • Use the ephemeral key to allow the user to select a payment method
  • Send to the BE using the corresponding url the payment method selected by the user (see docs)
  • Booking should now be in complete state
